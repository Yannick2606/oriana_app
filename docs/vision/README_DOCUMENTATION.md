# Documentation fondatrice orIAna

## Objectif

Ce dossier contient les documents de référence stratégique, fonctionnelle et méthodologique d'orIAna.

La documentation est considérée comme un actif du produit au même titre que le code.

## Organisation

- `ADN_ORIANA.md` : convictions fondatrices.
- `CONSTITUTION_ORIANA.md` : principes directeurs du projet.
- `METHODE_BOREAL.md` : cadre méthodologique propriétaire (nom provisoire).

## Règle de gouvernance

Les documents validés sont versionnés dans GitHub. Les évolutions importantes doivent être tracées et reliées aux décisions du projet.
